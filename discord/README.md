# Discord 数据目录

## 📁 目录结构

```
discord/
├── data/           # 数据存储
│   ├── products.json    # Product Hunt 产品数据
│   ├── news.json        # 新闻资讯数据
│   └── materials.json   # 公众号素材数据
```

### 日志记录
- 记录每次推送的内容
- 便于排查问题
- 监控任务运行状态
